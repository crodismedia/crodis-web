REGLA IMPORTANTE SOBRE LAS FOTOS
Las fotografías de páginas web, directorios, Google u otras fuentes públicas no se consideran
automáticamente libres para republicarlas. Las imágenes de la carpeta imagenes_publicas son
solo referencias para identificar el negocio. Antes de publicarlas en TallerMap debe solicitarse
autorización al titular del taller o utilizar una fotografía suministrada por él.

USO RECOMENDADO
1. Publicar inicialmente la imagen de imagenes_placeholder/ correspondiente al taller.
2. Contactar con el taller para confirmar todos los datos.
3. Solicitar logo y fotografía exterior en propiedad o con autorización.
4. Subir la foto autorizada a Supabase Storage.
5. Sustituir el nombre de archivo local por la URL pública de Supabase.
6. Cambiar activo/verificado únicamente después de la revisión.

AVISO SOBRE LOS DATOS
La información procede de páginas web y directorios públicos y no constituye un censo oficial.
No se han deducido correos ocultos. Los registros de calidad baja, antiguos o posibles duplicados
requieren una comprobación especial antes de publicarse.
